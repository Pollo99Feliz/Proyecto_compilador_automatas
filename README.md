# Proyecto_compilador_automatas
# CODEMODE Compiler

Este es un compilador educativo desarrollado en **Python** que utiliza una interfaz gráfica con **Tkinter**. El lenguaje permite la asignación de variables, operaciones matemáticas y salida de datos en pantalla con un fuerte enfoque en la validación semántica.

## Características
- **Resaltado de sintaxis:** Identifica tipos de datos en tiempo real.
- **Análisis Semántico:** Bloquea divisiones entre cero y operaciones inválidas.
- **Tabla de Símbolos:** Gestión de memoria persistente en archivos `.txt`.
- **Manejo de Errores:** Reportes detallados de línea y tipo de error.

## Cómo ejecutarlo
1. Asegúrate de tener instalado Python 3.x.
2. Descarga el archivo `main.py`.
3. Ejecuta en tu terminal: `python main.py`.
